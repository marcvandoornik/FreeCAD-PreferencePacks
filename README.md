# FreeCAD Preference Packs

My personal preference files for quickly bootstrapping new installations.

To initialize a nicely usable configuration directory:
- create the desired data directory
- issue `git clone https://github.com/marcvandoornik/FreeCAD-PreferencePacks .`
- edit initialize_addons.bat by removing REM from the desired git lines to install a starting set of workbenches
- run initialize_addons.bat
